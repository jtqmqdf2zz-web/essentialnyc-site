The doors in.

Each one starts where you already are — a single activity, a real concern — and opens onto the rest of the practice.

Section 01 — Three kinds door · synergy · system

Short ones to start. Longer ones to go deep.

Type I — the door. A few minutes on one activity. The model at a glance, one concrete loop to try.

Type II — the synergy. What happens when loops start feeding each other. Built like a piece of music: one voice, then another, then the whole thing playing.

Type III — the system. The full practice, end to end, when you're ready for it.
